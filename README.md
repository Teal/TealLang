Tea 语言是一个和 **运行效率接近C++、开发效率超越Java** 的编程语言。  

1. 编译为原生机器码（汇编），兼容各个操作系统（Windows, MacOS, Linux 等）和平台。
2. 现代化语言，支持 GC、面向对象、函数式编程、类型推导等流行语言特性。
3. 更实用，更适合敏捷开发。
4. 国产。
