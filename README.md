Tea
=========

Tea 语言的目标是创造一个 **运行效率接近C++、开发效率接近Python** 的编程语言。    
它可以编译生成支持各个平台、支持GC、不需要各种虚拟机的原生机器码，并且和C代码可以互相调用。
